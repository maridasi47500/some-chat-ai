Nos IA disponibles

Mistral AI propose plusieurs modèles d'IA, chacun optimisé pour des tâches spécifiques. Voici la liste des IA disponibles via notre API :

Mistral-General

Modèle polyvalent pour le texte, la conversation et la génération de contenu.

Mistral-Code

Spécialisé dans la génération et l'optimisation de code.

Mistral-Data

Base de données d'adresses et d'informations structurées.

Mistral-Social

Optimisé pour l'intégration sur les plateformes sociales.

Mistral-Travel

Dédié à la planification et l'organisation de voyages.

Mistral-Elsewhere

Modèle personnalisable pour des usages spécifiques.

Exemple d'utilisation de l'API

Voici comment importer et utiliser notre API en Python :

from mistral import MistralAPI
api = MistralAPI(api_key="votre_clé_api")
response = api.generate(model="Mistral-General", prompt="Bonjour, comment ça va ?")
print(response)

Vous pouvez instancier autant d'IA que nécessaire, selon vos besoins et votre quota API.
